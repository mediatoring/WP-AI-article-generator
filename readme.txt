Tento plugin generuje články a obrázky pomocí OpenAI GPT-4o-mini a DALL-E API. Plugin umožňuje automatické nebo manuální generování článků na základě specifikovaných kategorií a témat.

== Description ==

Generátor AI článků a obrázků je WordPress plugin, který využívá technologii OpenAI k automatickému generování obsahu. Plugin podporuje dvě hlavní funkce:
1. **Generování článků:** Vytváření textového obsahu na základě zvolených kategorií a témat.
2. **Generování obrázků:** Vytváření obrázků na základě textového promptu pomocí technologie DALL-E.

Plugin je ideální pro správce webových stránek, kteří chtějí automatizovat tvorbu obsahu a obohatit své články o unikátní obrázky.

== Installation ==

1. Nahrajte složku `generator-ai-articles-images` do adresáře `/wp-content/plugins/`.
2. Aktivujte plugin skrze menu `Pluginy` ve WordPressu.
3. Jděte do `Nastavení > Generátor článků` a nastavte své OpenAI API klíče.

== Usage ==

Po nastavení API klíčů:
1. Jděte na `Nastavení > Generátor článků`.
2. Vyberte, zda chcete povolit generování obrázků.
3. Nastavte požadovanou délku článku v tokenech.
4. Zvolte, zda má být článek publikován automaticky nebo jako koncept.

Můžete také využít manuální generování článků přímo z administrace WordPressu.

== Frequently Asked Questions ==

= Jaké jsou minimální požadavky na instalaci? =

WordPress 5.0 nebo vyšší, PHP 7.2 nebo vyšší.

= Jaké API klíče potřebuji? =

Budete potřebovat API klíč pro OpenAI, který získáte na webu OpenAI.

= Mohu nastavit délku generovaného článku? =

Ano, můžete specifikovat délku článku v tokenech v nastavení pluginu.

= Lze generovat i obrázky? =

Ano, pokud povolíte tuto možnost v nastavení, plugin bude generovat obrázky pomocí DALL-E.
